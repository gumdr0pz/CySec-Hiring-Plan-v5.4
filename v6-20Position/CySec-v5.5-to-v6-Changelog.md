# CySec Hiring Plan — v5.5 → v6 Change Log

**Date:** June 27, 2026
**Status:** Reference — what changed from v5.5 to the current v6

> v6 re-bases the plan on the company **HR job-level framework** and the **consultant Rate-card base salary** (per CTO/Dennis feedback), generalises support roles, and removes the COGS burden. Source of truth: `CySec-v6-Levels-Titles-Salary-Framework.md`.

---

## 1. Framework-level changes

| Aspect | v5.5 | v6 |
|--------|------|----|
| Role philosophy | Cyber-specific roles | **General roles** (dotted line to cyber) for support functions; cyber-core stays in the practice |
| Titles | Seniority labels (Sr/Jr/Lead/IC) | **Company-convention titles** (Engineer / Staff Engineer / Specialist / Team Lead / Technical Presales) |
| Job levels | Senior / Lead / Junior / IC | **HR job levels L1–L7**; Specialist = L4 (dual Specialist/Team Lead) |
| Salary basis | 30%-reduced bands (RM4,900–7,840 / RM3,500–5,600) + 1.44× COGS | **Rate-card "Rate" tab base benchmark**, no COGS multiplier |
| Wave structure | 7-5-8 (20 hires) | **7-5-7 (19 hires)** |
| Project/Product Manager | Junior Project Manager | No Project Manager and no standalone Product Manager. Only the cyber-core **Product Manager – Government Security** (L4). |
| Gov Sec role | Government Security Project Manager | **Product Manager – Government Security** (L4, cyber-core) |
| Leadership | "BU Head" / "Principal Architect" | **Head of Cybersecurity BU — Faurani** |
| Total cost | RM1,451,520 – 2,322,480 (12.1%–19.4%) | **RM1,908,000 – 2,712,000 (15.9%–22.6%)**, within 25% cap |
| COGS calculator | `cysec_cogs_calculator.py` | **Removed** |
